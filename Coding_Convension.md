Coding phải clean code, dễ đọc và tối ưu, đặc biệt phải bảo mật tránh các lỗi liên quan đến security, code phải tự giải thích và dễ hiểu

Tách component hợp lý tránh 1 file quá dài, có thể tái sử dụng code.

Code UIUX phải hiện đại, responsive cho các thiết bị.

- **Readability (Khả Năng Đọc)**: Code phải tự giải thích và dễ hiểu

- **Consistency (Tính Nhất Quán)**: Tuân theo các pattern đã được thiết lập trong toàn bộ codebase

- **Maintainability (Khả Năng Bảo Trì)**: Viết code dễ sửa đổi và mở rộng

- **Performance (Hiệu Suất)**: Xem xét các tác động về hiệu suất trong quyết định thiết kế

- **Security (Bảo Mật)**: Tuân theo các best practices về bảo mật

- **Internationalization (Đa Ngôn Ngữ)**: Tất cả text phải hỗ trợ đa ngôn ngữ
